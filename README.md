# Archip-RV32
Este é o primeiro processador RISC-V da Archip, sendo um processador de 32 bits focado em simplicidade e aprendizado, por causa disso o processador será simples mas irá ter varios comentarios para ajudar o entendimento. (Archip's first 32-bit RISC-V processor, focused on simplicity and learning. The code is heavily commented for educational purposes.)
